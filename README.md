#  Git To Work 

## 📂 프로젝트 소개
Git To Work는 개발자의 GitHub 활동 데이터와 최신 채용 정보를 결합하여,<br>
사용자의 기술 스택과 경력에 최적화된 직무 및 기업을 추천하는 맞춤형 모바일 앱입니다. <br> 

## 📅 프로젝트 수행 기간
- 2025-02-24 ~ 2025-04-11
